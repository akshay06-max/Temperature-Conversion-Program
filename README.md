# 🌡️ Temperature Conversion Program

A Python command-line program that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features
- Converts from any unit to the other two
- Input validation for non-numeric values
- Handles invalid Kelvin values (negative numbers)
- Clean, function-based structure

## How to Run

Make sure Python is installed, then run:

```bash
python temperature_converter.py
```
